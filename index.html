INTRODUCTION TO HTML

HTML is a computer mark up language that is used that is used for the creation of webpages. The letters HTML stand for Hyper Text Mark Up Language and its code is is used to establish the text and structure of webpages. The use of HTML enables a web developer to use a vast array of components ranging from text formatting such as making text bold or italic, to setting background images as a webpage background.
